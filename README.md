## 🚀 Transcript Summarizer

A lightweight transcript summarization tool built to generate high-quality datasets for Retrieval-Augmented Generation (RAG) pipelines.

Part of the **ChronoWeave** ecosystem, this tool converts raw transcripts into concise, structured summaries to improve LLM performance and retrieval accuracy.

---

## ✨ Features

- 🧠 Smart transcript summarization  
- ⚡ Handles long text efficiently  
- 📊 Generates structured outputs (JSON/CSV)  
- 🔗 Ready for RAG pipeline integration  

---

## ⚙️ Tech Stack

- Python  
- LLM APIs (OpenAI / Gemini / etc.)  
- JSON / CSV
